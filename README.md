### Set Up
1. Clone the repo
   ```sh
   git clone git@github.com:MattWalterTX/whats-cookin-project.git
   ```
2. Install NPM packages and start server
   ```sh
   npm install
   npm start
   ``` 
3. Enter the following url in your browser: http://localhost:8080/
4. Explore the website
